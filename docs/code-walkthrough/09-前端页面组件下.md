## `BlobTableViews.vue`（1537 行）— **最大页面**

建议分 **6 段**对照源码阅读：

### 段 1：imports 与 state（1-120 行）

| 行 | 说明 |
|----|------|
| 1-25 | Vue、router、API、工具 import |
| 30-65 | views、table、catalog、SQL 相关 ref |
| 57-73 | computed：当前 view、行预览项、**当前行在 tableRows 中的索引** |
| 84-110 | `browseContext`：SQL/浏览用数据源描述 |

### 段 2：目录树（122-235 行）

| 函数 | 说明 |
|------|------|
| `catalogNodeLabel` | 节点文案 `[表]`/`[数据库视图]` |
| `viewCatalogKey` / `catalogObjectKey` | 匹配已保存配置 |
| `findSavedViewForCatalogNode` | 树节点显示「已保存」 |
| `loadCatalogTree` | el-tree lazy：connection→database→object |
| `onCatalogNodeClick` | 选中对象；若有配置则打开 |

### 段 3：SQL 与创建配置（255-370 行）

| 函数 | 说明 |
|------|------|
| `openCreateViewDialog` / `submitCreateView` | 从目录创建浏览配置 |
| `handleSqlValidate` / `handleSqlExecute` | SQL 标签页 |

### 段 4：表格与行预览（371-680 行）

| 函数 | 说明 |
|------|------|
| `loadViews` / `loadRows` | 加载配置与分页数据 |
| `selectPreviewRow` / `goPrev/NextPreviewRow` | 在**全部已加载行**间切换；`↓` 到底自动 `loadMore` |
| `onPreviewKeydown` / `onPreviewWheel` | 键盘/滚轮（需先聚焦行预览面板） |
| `getRowKey` | `主键#行号` 保证 el-table 行唯一（避免重复 PK 导致高亮错乱） |
| `statusLabel` | `no_data` → 「无数据」；`pending` → 「未迁移」 |

### 段 5：生命周期与 watch（687-767 行）

| 块 | 说明 |
|----|------|
| `watch(activeViewId)` | 切换配置 reload rows |
| `watch(route.query.viewId)` | URL 深链 |
| `onMounted` | loadViews + 可选 loadRows |

### 段 6：template + style（768-1537 行）

| 块 | 说明 |
|----|------|
| 左栏 `catalog-tree-wrap` | **横向滚动**外层 |
| 右栏 tabs | 「表数据」+「SQL 查询」 |
| `table-panel` | 上表 + 下行预览 strip |
| `.row-preview-panel` | tabindex=0 接收键盘 |
| CSS | compact 行高、scrollbars |
